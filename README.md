# TaskAlarm

A smart alarm app that makes sure you actually wake up. Instead of hitting snooze while half-asleep, TaskAlarm requires you to complete short interactive tasks before the stop button unlocks.

## What It Does

TaskAlarm solves the classic "snooze addiction" problem. When your alarm rings, you must complete a set number of mini tasks to prove you're awake enough to stop the alarm.

**Core Flow:**
1. Alarm rings at scheduled time
2. Complete interactive tasks (math, color matching, shape matching, and more)
3. Stop button unlocks after all tasks are done
4. View an inspirational quote as you start your day

## Key Features

**Alarm Management**
- Create, edit, and delete alarms
- Weekday repeat schedules
- Enable/disable with one tap
- Custom alarm labels and sounds

**Task-Based Wake-Up System (3-12 tasks)**
- Math problems (addition, subtraction, multiplication)
- Color matching with visual color boxes
- Shape matching with rendered shapes
- Icon matching
- Position tapping
- Order tapping sequences
- Object counting
- Mixed mode (random selection)

**Reflection & Mindfulness**
- Optional reflection question as final task
- View and review past reflections
- Custom questions support (create your own Q&A tasks)

**Personalization**
- Light, dark, and system themes
- 12-hour and 24-hour time formats
- Custom vibration and ringtone per alarm
- Selectable default task types

**Privacy-First**
- All data stored locally on your device
- No account required
- No data collection or tracking
