# Screens

## Alarm List

- Header: `TaskAlarm`.
- Button: `+ New Alarm`.
- Alarm item list: time, on/off switch.

## Create/Edit Alarm : will open after clicking on set alarm

- Time picker.
- Weekday selector.
- Sound selector.
- Optional label.
- Advanced 
    - Vibration toggle.
    - Snooze policy (MVP): allowed only after required tasks are completed.
- Save action.

## Ringing Screen

- Full-screen takeover.
- Looping alarm sound.
- Task progress indicator.
- Active task card.
- No stop button until required tasks are done.

## Quote Screen (after stop)

- Success message.
- Random quote text.
- Optional action: close/back to list.

## Settings

- Default task count. Task count selector (`3` to `10`).
- Task type selector (`math`, `color`, `shape`, `mixed`).
- Snooze policy (`afterCompletionOnly` in MVP).
- Theme mode (`light`, `dark`, `system`).
