# User Flow

## Main flow

1. Open app.
2. Create alarm.
3. Select time, weekdays, sound, and task settings (one time setup).
4. Save alarm.
5. Alarm rings at scheduled time.
6. Ringing screen opens with first task.
7. User completes required tasks.
8. Stop unlocks.
9. User taps stop.
10. Show random quote of the day.

## Simple task flow

- Show one task at a time.
- Update progress (example: `2/4`).
- Unlock stop only when progress reaches target.

## Quote flow (MVP)

- Use a local quote list stored in app data or use AI model.
- Pick one quote randomly after alarm stop.
- Display quote on a small success screen/modal.
